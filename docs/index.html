Километраж

Маршрут

Автоматично: Маршрут 1 = Изминати км / 2 и Маршрут 2 = Изминати км / 2.

Гориво

Как се изчисляват стойностите:

  • Количество = Наличност + Доливано 1 + Доливано 2
  • Остатък = попълва се ръчно (реално измерен)
  • Икономия = (Очакван разход) - (Реален разход)
  • Реален разход = Количество - Остатък
    Очакван разход = Изминати км × 0.1 л/км
